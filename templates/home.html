About us

Terra is a simple waste disposal solution that connects the public with the authorities. It is a platform to promote the idea of proper waste disposal.

THE TERRA APP

The Terra Application, currently only for Android Phones, will be available to all users and provide a feature for users to upload photographs for immediate pick-ups.

Features Offered

TERRA offers the following features for users:

  • PHOTOGRAPH

    The Registered and Un-Registered users can upload a photo of a pile of collected garbage.

  • USERS

    Registered and Un-Registered Users will be able to use the system.

  • AREA ADMIN

    There will different administrators assigned to each different locality.

  • SCHEDULING

    Registered users can use the application or website to schedule pick-ups periodically.

  • DRIVERS

    Drivers will be assigned pick-up duties for both scheduled and unregisterd users.

  • MOBILE APPLICATION

    The android application will be available to drivers and users to use the features offered by TERRA
